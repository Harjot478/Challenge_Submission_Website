# Student Challenge Platform

## Project Overview
This project is a simple challenge platform where companies can post design challenges and students can submit their work. The platform allows challenges to be reviewed, rated, and winners to be selected and showcased.

## Features
- Companies can submit challenges using a form
- Students can view available challenges
- Students can submit their work through a submission form
- Submissions are collected automatically
- Winners are selected and displayed on the website

## Tools Used
- HTML
- CSS
- JavaScript
- GitHub Pages
- Google Forms
- Google Sheets

## How It Works
Companies submit challenges through a Google Form. Students view the challenges on the website and submit their work using another Google Form. Submissions are reviewed and winners are chosen and displayed on the Past Winners page.
